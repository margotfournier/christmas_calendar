# Logos

Ce dossier contient les logos de HomeExchange et Notion.

## Comment ajouter les logos

1. Placez vos fichiers de logos dans ce dossier (`public/logos/`)
   - Exemple : `homeexchange-logo.svg` ou `homeexchange-logo.png`
   - Exemple : `notion-logo.svg` ou `notion-logo.png`

2. Utilisez-les dans votre code avec le chemin `/logos/nom-du-fichier.ext`
   - Les fichiers dans `public/` sont servis à la racine du site
   - Exemple : `<img src="/logos/homeexchange-logo.svg" alt="HomeExchange" />`

## Formats recommandés

- **SVG** : Meilleure qualité, scalable, léger (recommandé)
- **PNG** : Pour les logos avec transparence
- **WebP** : Format moderne et optimisé
